# Chrome Web Store Listing

Use this copy when publishing or updating the Chrome Web Store listing. The summary comes from `manifest.json`; the description belongs in the Chrome Web Store dashboard.

## Summary

Work faster in Vantage with a clearer timesheet, smarter entry tools, and helpful error checks.

## Description

ad-vantage helps State of Utah employees spend less time navigating the Vantage timesheet and more time completing it accurately. It improves the wide, data-heavy grids in Vantage without changing the underlying timesheet workflow.

Keep the details you need in view by freezing important columns and hiding the ones that get in the way. Fetch task lookup data from ServiceNow to show matching descriptions and get type-ahead suggestions while entering Daily Activity codes. Your synced lookup data stays in your current Chrome profile.

The extension also streamlines common time-entry work: it selects the largest available page size for grids with more than 20 rows, provides one-click access to Update Timesheet on the Daily Activity tab, and flags time entries that are not on a quarter-hour increment or have missing Event values.
